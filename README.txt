פרויקט Fullstack Development
סקירה כללית
פרויקט זה מכיל מספר תתי-פרויקטים המדגימים יכולות שונות באמצעות HTML, CSS, ו-JavaScript. הפרויקט מחולק לשני חלקים עיקריים: דפי נחיתה (Landing Pages) באמצעות HTML ו-CSS, ופרויקטים ב-JavaScript המדגימים OOP, שימוש ב-API, ו-ES6.
